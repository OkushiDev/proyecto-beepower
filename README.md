# Proyecto BeePower

## Descripción
Plataforma web en desarrollo. Este repositorio implementa una arquitectura de monorepositorio para gestionar el cliente y el servidor, con infraestructura orquestada mediante contenedores.

## Pila Tecnológica
* **Frontend:** TypeScript
* **Backend:** FastAPI (Python)
* **Base de Datos:** PostgreSQL
* **Infraestructura y Orquestación:** Docker, Docker Compose

## Estructura del Directorio
* `/frontend`: Código fuente del cliente web.
* `/backend`: Código fuente de la API y lógica de servidor.
